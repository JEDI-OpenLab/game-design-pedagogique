# Exemple : L'Odyssée de l'enseignement

## Type

Livre-jeu pédagogique / histoire dont vous êtes le héros.

## Source

Version web existante : `JEDI-OpenLab/livre-jeu-odyssee-enseignement-main`.

## Intérêt pédagogique

Cet exemple montre comment un récit à embranchements peut soutenir :

- l'immersion narrative ;
- l'individualisation des parcours ;
- l'essai-erreur ;
- la prise de décision ;
- la métacognition ;
- le lien entre règles d'écriture et design pédagogique.

## Mécaniques principales

- choix à conséquences ;
- chapitres reliés ;
- feedback narratif ;
- possibilité de rejouer ;
- progression visuelle ;
- ressources pédagogiques associées.

## À documenter ensuite

- carte complète du parcours ;
- matrice objectifs pédagogiques / choix narratifs ;
- règles d'écriture ;
- retours de test ;
- critères de réutilisation dans d'autres cours.
