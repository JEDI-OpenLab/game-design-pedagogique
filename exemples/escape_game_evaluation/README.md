# Exemple : escape game sur l'évaluation

## Type

Escape game pédagogique appliqué aux pratiques d'évaluation.

## Source

Dossier `Escape_Game/Escape Game - Evaluation` dans `01_Projets/Jeux_serieux`.

## Intérêt pédagogique

Cet exemple montre comment faire manipuler des notions d'évaluation par l'enquête, l'énigme et la coopération.

## Mécaniques principales

- mission narrative ;
- documents à analyser ;
- énigmes ;
- indices progressifs ;
- temps limité ;
- rôle du maître du jeu ;
- débriefing.

## Points à documenter ensuite

- synopsis ;
- objectifs pédagogiques ;
- ordre ou réseau d'énigmes ;
- fiches de synthèse utilisées ;
- modalités de débriefing ;
- droits de diffusion des supports sources.
