# Gabarit de cartes questions

## Carte simple

```text
Catégorie :
Niveau d'apprentissage :
Question :
Réponse attendue :
Feedback :
Source / ressource :
Variante ou indice :
```

## Niveaux conseillés

| Niveau | Intention | Exemple de formulation |
| --- | --- | --- |
| 1 - Connaître | restituer une notion | Définir, citer, nommer, identifier |
| 2 - Comprendre | expliquer ou associer | Reformuler, classer, expliquer, comparer |
| 3 - Appliquer | utiliser dans un cas | Choisir, appliquer, résoudre, produire |

## Conseils

- Une carte doit pouvoir être lue rapidement.
- Le feedback doit expliquer, pas seulement valider.
- Les réponses doivent être vérifiables.
- Les cartes de niveaux différents doivent mobiliser des opérations cognitives différentes.
