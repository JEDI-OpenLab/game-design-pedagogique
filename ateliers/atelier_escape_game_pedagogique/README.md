# Atelier : créer un escape game pédagogique

## Intention

Concevoir une activité coopérative où les apprenants résolvent des énigmes alignées avec des objectifs pédagogiques dans un cadre narratif et temporel.

## Principe

Un escape game pédagogique combine :

- une mission ;
- un espace physique, numérique ou hybride ;
- des énigmes ;
- des indices ;
- une contrainte de temps ;
- un débriefing.

## Objectifs

À l'issue de l'atelier, les participants sont capables de :

- définir le rôle pédagogique d'une énigme ;
- concevoir une progression d'indices ;
- éviter les blocages destructeurs ;
- préparer le rôle du maître du jeu ;
- débriefer les apprentissages.

## Déroulé type

| Étape | Activité | Durée |
| --- | --- | --- |
| Cadrage | objectifs et public | 20 min |
| Mission | contexte narratif et but collectif | 20 min |
| Énigmes | conception de 2 à 4 énigmes alignées | 60 min |
| Indices | paliers d'aide et remédiation | 30 min |
| Prototype | supports, codes, documents, minuteur | 45 min |
| Test | résolution par un autre groupe | 30 min |
| Débriefing | apprentissages et améliorations | 25 min |

## Conseils

- Une énigme doit faire manipuler le contenu d'apprentissage.
- Prévoir plusieurs niveaux d'indices.
- Le maître du jeu accompagne sans résoudre.
- Le temps limité crée de la tension, mais ne doit pas empêcher d'apprendre.
- Le débriefing permet de sortir du jeu et de formaliser les acquis.

## Supports possibles

- Genially en navigation microsite ;
- documents papier ;
- cadenas réels ou virtuels ;
- QR codes ;
- faux documents ;
- cartes indices ;
- grille de synthèse.
