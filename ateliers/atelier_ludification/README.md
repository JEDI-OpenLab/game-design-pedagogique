# Atelier : concevoir une activité ludifiée

## Intention

Ajouter des mécanismes ludiques à une activité pédagogique existante en conservant l'alignement avec les apprentissages visés.

## Objectifs

À l'issue de l'atelier, les participants sont capables de :

- identifier un problème d'engagement ou de progression ;
- choisir des leviers ludiques adaptés ;
- construire une boucle d'activité-feedback ;
- vérifier que la ludification n'ajoute pas de bruit ;
- préparer un test rapide.

## Déroulé type

| Étape | Activité | Durée |
| --- | --- | --- |
| Diagnostic | activité actuelle, irritants, objectifs | 30 min |
| Leviers | progression, défi, coopération, narration, feedback | 30 min |
| Scénarisation | parcours et moments clés | 45 min |
| Prototype | supports et règles simples | 45 min |
| Test | simulation courte | 20 min |
| Débriefing | alignement et ajustements | 20 min |

## Leviers fréquents

- progression visible ;
- défis courts ;
- feedbacks immédiats ;
- coopération ;
- choix et autonomie ;
- collection ou inventaire ;
- narration légère ;
- niveaux de difficulté.

## Vigilance

La ludification ne doit pas maquiller une activité mal cadrée. Elle doit résoudre un problème précis : manque de feedback, faible engagement, difficulté à suivre la progression, besoin d'entraînement, besoin de coopération.
