# Atelier : créer un serious game narratif

## Intention

Faire concevoir un parcours narratif à embranchements pour transformer une situation d'apprentissage en expérience active, immersive et feedbackée.

## Inspiration source

Atelier `Jouer avec ses méthodes d’enseignement - créer un serious game`, appuyé sur Moodle Leçon et le scénario `Une journée normale à Centrale Lille`.

## Public cible

Enseignants, enseignants-chercheurs, ingénieurs pédagogiques, formateurs.

## Durée indicative

2 h à 3 h pour une initiation. Une demi-journée pour produire un prototype testable.

## Objectifs

À l'issue de l'atelier, les participants sont capables de :

- distinguer un jeu narratif pédagogique d'une simple histoire interactive ;
- formuler les objectifs pédagogiques d'un parcours à choix ;
- écrire une micro-situation avec choix, feedback et conséquence ;
- prototyper un embranchement simple ;
- identifier les conditions de test et d'amélioration.

## Déroulé

| Étape | Activité | Durée |
| --- | --- | --- |
| Introduction | objectifs, exemples, vocabulaire | 15 min |
| Cadrage | public, besoin, objectifs d'apprentissage | 20 min |
| Scénarisation | personnage, situation, perturbation, choix | 35 min |
| Prototypage | écrire 3 à 5 écrans ou cartes | 45 min |
| Test croisé | faire jouer un autre groupe | 25 min |
| Débriefing | feedbacks, règles, suite | 20 min |

## Consignes de conception

- Utiliser l'adresse directe : "vous".
- Donner au joueur un rôle clair.
- Chaque choix doit produire une conséquence.
- Les erreurs doivent mener à une remédiation ou à une conséquence utile.
- Éviter les impasses qui cassent l'immersion.

## Livrables

- fiche de conception ;
- carte du parcours ;
- premier scénario ;
- liste des feedbacks ;
- grille de test.

## Variantes

- Moodle Leçon pour un prototype numérique.
- Twine ou simple fichier Markdown pour une version légère.
- Cartes papier si l'objectif est d'abord de tester la logique narrative.
