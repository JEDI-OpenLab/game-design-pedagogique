# Évaluer et améliorer

L'évaluation d'un jeu pédagogique porte sur l'apprentissage, la jouabilité et l'utilité de l'activité dans son contexte.

## Trois niveaux

### Apprentissage

- Les objectifs sont-ils atteints ?
- Quelles connaissances ou compétences sont mobilisées ?
- Les apprenants savent-ils expliquer ce qu'ils ont appris ?

### Expérience de jeu

- Les règles sont-elles comprises ?
- Le défi est-il équilibré ?
- Les feedbacks sont-ils utiles ?
- Les joueurs restent-ils engagés ?

### Dispositif pédagogique

- L'activité tient-elle dans le temps prévu ?
- L'animation est-elle réaliste ?
- Le débriefing fonctionne-t-il ?
- Les traces sont-elles exploitables ?

## Indicateurs possibles

- réponses correctes ou justifiées ;
- qualité des productions ;
- décisions prises dans le jeu ;
- verbalisations pendant le débriefing ;
- observation des interactions ;
- questionnaire court après activité ;
- comparaison entre version initiale et version améliorée.

## Amélioration continue

Après chaque test, décider explicitement :

- ce qui reste inchangé ;
- ce qui est simplifié ;
- ce qui est supprimé ;
- ce qui est ajouté ;
- ce qui doit être testé à nouveau.
