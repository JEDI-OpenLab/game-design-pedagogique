# Choisir le type de jeu

Le type de jeu se choisit à partir des objectifs pédagogiques, du public, des contraintes et du type d'expérience souhaitée.

## Repère par objectifs

| Objectif dominant | Types de jeux adaptés | Points de vigilance |
| --- | --- | --- |
| Mémoriser, restituer | cartes questions, quiz scénarisé, plateau simple | éviter le simple QCM décoré |
| Comprendre, expliquer | jeu d'association, cas d'usage, jeu de rôle court | rendre le feedback explicatif |
| Appliquer une procédure | parcours, simulation, escape game, livre-jeu | prévoir indices et remédiations |
| Analyser une situation | enquête, jeu de rôle, simulation, débat ludifié | garder des traces pour le débriefing |
| Évaluer, arbitrer | jeu de décision, dilemme, simulation collective | expliciter les critères |
| Créer | production constructionniste, prototype de jeu, cartes à produire | valoriser la démarche, pas seulement l'objet final |

## Repère par contraintes

- Temps court : cartes, quiz scénarisé, mini-défi, mécanique de choix.
- Demi-journée : dégamification, construction de prototype, escape game court.
- Autoformation : livre-jeu, parcours à embranchements, simulation guidée.
- Présentiel collectif : plateau, jeu de rôle, escape game, activité de construction.
- Peu de matériel : cartes papier, scénario Markdown, Moodle Leçon, Genially, diaporama interactif.

## Repère par public

Un public débutant a besoin de règles très simples, de feedbacks fréquents et d'un accompagnement. Un public expert accepte mieux l'ambiguïté, la simulation complexe et les dilemmes.

## Critère de décision

Le bon type de jeu est celui qui rend l'activité d'apprentissage plus claire, plus engageante ou plus formatrice qu'une modalité non ludique équivalente.
