# Construire scénario, règles et feedbacks

Un jeu pédagogique tient sur trois éléments solidaires : une situation, des règles d'action et des feedbacks interprétables.

## Scénario

Le scénario donne du sens à l'action. Il peut être très simple : une mission, une enquête, une journée type, une situation professionnelle, un défi collectif.

Pour un jeu narratif, une micro-situation peut suivre cette structure :

1. Situation initiale.
2. Élément perturbateur.
3. Choix ou action.
4. Résolution.
5. Conséquence ou transition.

## Règles

Les règles doivent permettre de jouer rapidement. Une règle pédagogique utile :

- définit ce que le joueur peut faire ;
- produit un feedback ;
- aide à atteindre ou vérifier l'objectif ;
- reste compréhensible sans longue explication.

## Feedbacks

Le feedback est le moteur de l'apprentissage. Il peut être :

- correctif : bonne/mauvaise réponse ;
- explicatif : pourquoi cela fonctionne ;
- narratif : conséquence dans l'histoire ;
- stratégique : conseil pour rejouer ;
- collectif : discussion pendant le débriefing.

## Règle d'écriture

Une action de joueur devrait produire au moins une conséquence lisible :

```text
Le joueur fait X.
Le jeu répond Y.
L'apprenant comprend Z.
```

Si le Z reste flou, le feedback doit être réécrit.
