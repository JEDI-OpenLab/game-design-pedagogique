# Distinguer les notions clés

Le vocabulaire du jeu pédagogique mélange souvent des notions proches. Les distinguer évite de concevoir un dispositif séduisant mais mal aligné.

## Serious game

Un serious game est un jeu conçu ou adapté pour poursuivre une finalité autre que le divertissement seul : apprendre, s'entraîner, sensibiliser, coopérer, décider, simuler, etc.

Dans un cadre pédagogique, il devient pertinent lorsque ses règles, supports, défis et feedbacks soutiennent directement les objectifs d'apprentissage.

## Serious gaming

Le serious gaming consiste à utiliser un jeu existant dans un contexte sérieux, sans forcément modifier le jeu lui-même.

Exemple : utiliser un jeu de gestion ou de coopération disponible dans le commerce pour faire analyser des stratégies, des interactions ou des modèles de décision.

## Ludification ou gamification

La ludification consiste à ajouter des éléments issus du jeu à une activité non ludique : progression, badges, défis, avatar, narration, feedbacks, niveaux, collection, coopération, etc.

La vigilance principale : un élément ludique ne devient pédagogique que s'il sert une intention d'apprentissage.

## Dégamification

La dégamification consiste à retirer, simplifier ou modifier des éléments d'un jeu existant pour les recentrer sur une finalité pédagogique.

Dans un Trivial Pursuit pédagogique, par exemple, on peut supprimer des règles qui ralentissent la partie ou détournent l'attention, puis transformer les cartes en questions de connaissance, compréhension et application.

## Jeu éducatif et jeu pédagogique

Un jeu éducatif peut viser un apprentissage large, parfois informel. Un jeu pédagogique est conçu pour un contexte d'enseignement ou de formation, avec des objectifs explicites, des consignes, une animation et une évaluation.

## Constructionnisme

Une activité peut aussi faire apprendre en demandant aux participants de construire un jeu. Le produit final compte moins que la conceptualisation nécessaire pour le produire : sélectionner les contenus, formuler des questions, concevoir des règles, tester, corriger.
