# Animer un atelier

Un atelier de game design pédagogique doit produire deux choses : un prototype et une compréhension plus fine de l'alignement pédagogique.

## Déroulé générique

1. Cadrer le besoin et les objectifs.
2. Clarifier les notions clés.
3. Choisir ou analyser une mécanique de jeu.
4. Concevoir un prototype.
5. Tester rapidement.
6. Débriefer.
7. Décider des améliorations.

## Rôle de l'animateur

L'animateur garde le cap pédagogique. Il aide le groupe à ne pas se perdre dans la décoration, la complexité ou la nostalgie du jeu de référence.

## Matériel type

- fiches de conception ;
- cartes objectifs ;
- cartes mécaniques ;
- gabarits de règles ;
- papier, feutres, ciseaux, dés, pions ;
- grille de test ;
- minuteur ;
- support de débriefing.

## Points de vigilance

- Commencer par les objectifs, pas par le thème.
- Limiter volontairement le nombre de règles.
- Prévoir un test même très court.
- Débriefer l'apprentissage, pas seulement l'expérience de jeu.
