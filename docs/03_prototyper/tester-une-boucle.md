# Tester une boucle de jeu

Le test d'un jeu pédagogique ne vise pas seulement à savoir si les participants ont aimé. Il sert à comprendre ce que le jeu leur fait faire, apprendre et verbaliser.

## Préparer le test

Avant le test, définir :

- le public test ;
- la durée ;
- la boucle à observer ;
- les objectifs pédagogiques concernés ;
- les points sur lesquels l'animateur n'intervient pas ;
- les questions de débriefing.

## Observer

Pendant la partie, noter :

- les incompréhensions de règles ;
- les blocages ;
- les détours inutiles ;
- les moments d'engagement ;
- les stratégies des joueurs ;
- les feedbacks ignorés ou mal compris ;
- les traces d'apprentissage.

## Débriefer

Questions utiles :

- Qu'avez-vous essayé de faire ?
- Qu'est-ce qui vous a aidé à progresser ?
- Qu'est-ce qui vous a bloqué ?
- Qu'avez-vous compris ou révisé ?
- Quelle règle changeriez-vous ?
- Quel feedback manquait ?

## Décider

Classer les retours en quatre catégories :

- règles à clarifier ;
- contenus à corriger ;
- mécaniques à simplifier ;
- objectifs à réaligner.
